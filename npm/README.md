# eburon-codemaxxx

CodeMaxxx — AI coding agent for the terminal, powered by Ollama.

## Install

```bash
npm i -g eburon-codemaxxx
```

## Usage

```bash
codemax                        # launch TUI agent
codemax install                # install Ollama + pull model + OpenCode
codemax launch                 # launch via ollama launch
codemax tui                    # launch TUI agent
codemax help                   # show help
eburon-codemaxxx               # compatibility alias
```

## Other install methods

```bash
# curl one-liner
curl -fsSL https://raw.githubusercontent.com/eburondeveloperph-gif/probable-tribble/main/setup.sh | bash

# Homebrew
brew tap eburondeveloperph-gif/codemaxxx
brew install eburon-codemaxxx
```
